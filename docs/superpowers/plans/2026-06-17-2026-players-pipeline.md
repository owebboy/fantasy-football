# 2026 Player Data Pipeline Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace 2025 fantasy player data with 2026 rankings via a modular, re-runnable data pipeline (ESPN user-provided parser, FleaFlicker API client, FantasyPros Cheerio scraper → merge → transform → players.ts).

**Architecture:** Three independent data source modules feed a cross-reference merge script, which produces `merged-all.json`. The existing `transform-data.js` (modified for ESPN-primary sort + 2026 bye weeks) consumes that and generates `src/players.ts`. The Svelte app is unchanged.

**Tech Stack:** TypeScript (tsx runtime), Node.js, Cheerio for HTML parsing, FleaFlicker REST API, existing transform-data.js (Node).

### Task 3: FantasyPros Cheerio Scraper

**Files:**
- Create: `src/data/scrapers/fantasypros.ts`

The FantasyPros consensus rankings page is server-rendered at `https://www.fantasypros.com/nfl/rankings/ppr-cheatsheets.php`. We parse the player rows with Cheerio.

- [ ] **Step 1: Create the FantasyPros scraper**

`src/data/scrapers/fantasypros.ts`:

```ts
import * as cheerio from "cheerio";
import type { ScrapedPlayer } from "../types";
import { normalizeTeam, writeCache } from "./utils";

const URL = "https://www.fantasypros.com/nfl/rankings/ppr-cheatsheets.php";

export async function scrape(): Promise<ScrapedPlayer[]> {
  console.log("[fantasypros] Fetching rankings...");
  const resp = await fetch(URL, {
    headers: {
      "User-Agent": "Mozilla/5.0 (X11; Linux x86_64) AppleWebKit/537.36",
    },
  });
  if (!resp.ok) throw new Error(`FantasyPros fetch failed: ${resp.status}`);

  const html = await resp.text();
  const $ = cheerio.load(html);
  const players: ScrapedPlayer[] = [];

  // FantasyPros uses a table with player rows. Try multiple selector strategies.
  // The main table has rows with data-column attributes for rank, name, team, position.
  $("tr").each((_, row) => {
    const cells = $(row).find("td");
    if (cells.length < 4) return;

    const rankText = $(cells[0]).text().trim();
    const rank = parseInt(rankText, 10);
    if (isNaN(rank)) return;

    // Name is typically in a player-name link
    const nameEl = $(cells[1]).find("a.player-name, a.fp-player-link");
    const name = (nameEl.length ? nameEl.text().trim() : $(cells[1]).text().trim()).replace(/\s+/g, " ");

    // Team is in a separate cell, usually abbreviated
    const teamRaw = $(cells[2]).text().trim();
    const team = normalizeTeam(teamRaw);

    // Position
    const position = $(cells[3]).text().trim().toUpperCase();

    if (name && team && position) {
      players.push({ name, team, position, rank });
    }
  });

  // If the primary selector didn't work, try alternate parsing
  if (players.length === 0) {
    console.log("[fantasypros] Primary selector found 0 rows, trying fallback...");
    // Try finding a script tag with embedded JSON (some FP pages have this)
    const scripts = $("script").toArray();
    for (const script of scripts) {
      const text = $(script).html() || "";
      // Look for ecrData or playerData JSON patterns
      const match = text.match(/(?:ecrData|playerData|rankingsData)\s*=\s*(\[[\s\S]*?\]);/);
      if (match) {
        const data = JSON.parse(match[1]);
        for (const item of data) {
          if (item.player_name && item.rank_ecr) {
            players.push({
              name: item.player_name,
              team: normalizeTeam(item.player_team_id || item.team || ""),
              position: (item.player_position_id || item.position || "").toUpperCase(),
              rank: parseInt(item.rank_ecr, 10),
            });
          }
        }
        break;
      }
    }
  }

  console.log(`[fantasypros] Extracted ${players.length} players`);
  return players;
}

// Run directly
const isMain = process.argv[1]?.endsWith("fantasypros.ts") || process.argv[1]?.endsWith("fantasypros.js");
if (isMain) {
  scrape()
    .then((players) => {
      writeCache("fantasypros", players);
      console.log(`[fantasypros] Cache written: ${players.length} players`);
    })
    .catch((err) => {
      console.error("[fantasypros] Failed:", err.message);
      process.exit(1);
    });
}
```

### Task 5: ESPN Auto-Detect Parser

**Files:**
- Create: `src/data/scrapers/espn.ts`

This module reads any user-provided file at `data/cache/espn-input.*` (CSV, TSV, JSON, or plain text) and auto-detects the format to extract player data.

- [ ] **Step 1: Create the ESPN parser**

`src/data/scrapers/espn.ts`:

```ts
import fs from "fs";
import path from "path";
import type { ScrapedPlayer } from "../types";
import { normalizeTeam, writeCache } from "./utils";

const INPUT_GLOB = "data/cache/espn-input";

function findInputFile(): string | null {
  const exts = [".csv", ".tsv", ".json", ".txt", ".html"];
  for (const ext of exts) {
    const p = INPUT_GLOB + ext;
    if (fs.existsSync(p)) return p;
  }
  return null;
}

function parseCSV(content: string): ScrapedPlayer[] {
  const lines = content.split("\n").filter((l) => l.trim());
  const players: ScrapedPlayer[] = [];

  // Detect delimiter: comma or tab
  const delimiter = content.includes("\t") ? "\t" : ",";

  for (const line of lines) {
    const cols = line.split(delimiter).map((c) => c.trim().replace(/^"|"$/g, ""));
    // Try: rank, name, team, position (4 cols)
    if (cols.length >= 3) {
      const rank = parseInt(cols[0], 10);
      if (!isNaN(rank) && cols[1]) {
        players.push({
          rank,
          name: cols[1],
          team: normalizeTeam(cols[2] || ""),
          position: (cols[3] || "").toUpperCase(),
        });
      }
    }
    // Try: name, team, position, rank (different order)
    else if (cols.length === 3 && !isNaN(parseInt(cols[2], 10))) {
      players.push({
        rank: parseInt(cols[2], 10),
        name: cols[0],
        team: normalizeTeam(cols[1]),
        position: "",
      });
    }
  }

  // If header row detected (first row has "rank" or "name"), skip it
  if (players.length > 0 && /rank|name|player/i.test(lines[0].toLowerCase())) {
    players.shift();
  }

  return players;
}

function parseJSON(content: string): ScrapedPlayer[] {
  const data = JSON.parse(content);
  const arr = Array.isArray(data) ? data : data.players || data.data || [];

  return arr.map((item: any, idx: number) => ({
    name: item.name || item.player_name || item.playerName || "",
    team: normalizeTeam(item.team || item.team_abbreviation || item.proTeam || ""),
    position: (item.position || item.pos || "").toUpperCase(),
    rank: parseInt(item.rank || item.overallRank || item.ranking, 10) || idx + 1,
  }));
}

function parseText(content: string): ScrapedPlayer[] {
  // One player per line: "1. Ja'Marr Chase, CIN, WR"
  const lines = content.split("\n").filter((l) => l.trim());
  const players: ScrapedPlayer[] = [];

  for (const line of lines) {
    // Pattern: "1. Name, TEAM, POS" or "1 Name (TEAM) - POS"
    const match = line.match(/^(\d+)[\.\)]\s*(.+?)(?:\s*[,\(]\s*([A-Z]{2,3})\s*[,\)]\s*(\w+))?/);
    if (match) {
      const rank = parseInt(match[1], 10);
      let name = match[2].trim();
      let team = match[3] || "";
      let position = match[4] || "";

      // If no explicit team/pos, try to extract from name suffix like "Name, CIN WR"
      const restMatch = name.match(/^(.+?)[,\(]\s*([A-Z]{2,3})\s+(\w+)\s*[\),]?$/);
      if (restMatch && !team) {
        name = restMatch[1].trim();
        team = restMatch[2];
        position = restMatch[3];
      }

      if (name) {
        players.push({
          rank,
          name,
          team: normalizeTeam(team),
          position: position.toUpperCase(),
        });
      }
    }
  }

  return players;
}

export async function scrape(): Promise<ScrapedPlayer[]> {
  console.log("[espn] Looking for user-provided input file...");
  const inputPath = findInputFile();

  if (!inputPath) {
    console.warn("[espn] No input file found at data/cache/espn-input.*");
    console.warn("[espn] Drop your ESPN rankings file there (any format) and re-run.");
    return [];
  }

  console.log(`[espn] Reading ${inputPath}...`);
  const content = fs.readFileSync(inputPath, "utf8");
  const ext = path.extname(inputPath).toLowerCase();

  let players: ScrapedPlayer[] = [];

  // Auto-detect format and parse
  if (ext === ".json") {
    players = parseJSON(content);
  } else if (ext === ".csv" || ext === ".tsv") {
    players = parseCSV(content);
  } else if (ext === ".txt" || ext === ".html") {
    // Try text first, fall back to CSV
    players = parseText(content);
    if (players.length === 0) {
      players = parseCSV(content);
    }
    // If HTML, try extracting from table as last resort
    if (players.length === 0 && ext === ".html") {
      const tdMatch = content.match(/<td[^>]*>(\d+)<\/td>\s*<td[^>]*>([^<]+)<\/td>\s*<td[^>]*>([^<]+)<\/td>\s*<td[^>]*>([^<]+)<\/td>/gi);
      // Basic HTML table extraction fallback
    }
  } else {
    // Unknown extension — try all parsers
    players = parseCSV(content);
    if (players.length === 0) players = parseText(content);
    if (players.length === 0) {
      try { players = parseJSON(content); } catch {}
    }
  }

  console.log(`[espn] Parsed ${players.length} players from user input`);
  return players;
}

const isMain = process.argv[1]?.endsWith("espn.ts") || process.argv[1]?.endsWith("espn.js");
if (isMain) {
  scrape()
    .then((players) => {
      if (players.length > 0) {
        writeCache("espn", players);
        console.log(`[espn] Cache written: ${players.length} players`);
      } else {
        console.log("[espn] No players to write. Provide input file at data/cache/espn-input.*");
      }
    })
    .catch((err) => {
      console.error("[espn] Failed:", err.message);
      process.exit(1);
    });
}
```

### Task 7: Merge Script

**Files:**

- [ ] **Step 1: Create the merge script**

`src/data/merge.ts`:

```ts
import fs from "fs";
import { readCache } from "./scrapers/utils";
import type { ScrapedPlayer, CacheFile } from "./types";

interface MergedEntry {
  name: string;
  team: string;
  position: string;
  matchConfidence: number;
  status: string;
  sources: string[];
  espnRank: number | null;
  espnId: string | null;
  fleaflickerRank: number | null;
  fleaflickerId: number | null;
  fantasyprosRank: number | null;
  fantasyprosId: string | null;
  averageRank: number;
  rankVariance: number;
}

function normalizeName(name: string): string {
  return name
    .toLowerCase()
    .replace(/[^a-z0-9]/g, "")
    .replace(/jr$|sr$|iii$|ii$|iv$|v$/i, "")
    .trim();
}

function merge(): MergedEntry[] {
  const espn = readCache("espn");
  const ff = readCache("fleaflicker");
  const fp = readCache("fantasypros");

  if (!espn && !ff && !fp) {
    console.error("No cache files found. Run data:refresh first.");
    process.exit(1);
  }

  // Build a master index by normalized name
  const master: Map<string, MergedEntry> = new Map();

  function getOrCreate(key: string, name: string, team: string, position: string): MergedEntry {
    if (!master.has(key)) {
      master.set(key, {
        name,
        team,
        position,
        matchConfidence: 0,
        status: "unmatched",
        sources: [],
        espnRank: null,
        espnId: null,
        fleaflickerRank: null,
        fleaflickerId: null,
        fantasyprosRank: null,
        fantasyprosId: null,
        averageRank: 0,
        rankVariance: 0,
      });
    }
    return master.get(key)!;
  }

  function addSource(cache: CacheFile | null, rankField: string, idField: string, sourceName: string) {
    if (!cache) return;
    for (const p of cache.players) {
      const key = normalizeName(p.name);
      const entry = getOrCreate(key, p.name, p.team, p.position);
      (entry as any)[rankField] = p.rank;
      (entry as any)[idField] = (p as any).id || null;
      if (!entry.sources.includes(sourceName)) {
        entry.sources.push(sourceName);
      }
      if (!entry.team || entry.team.length < 2) {
        entry.team = p.team;
      }
      if (!entry.position) {
        entry.position = p.position;
      }
    }
  }

  addSource(espn, "espnRank", "espnId", "espn");
  addSource(ff, "fleaflickerRank", "fleaflickerId", "fleaflicker");
  addSource(fp, "fantasyprosRank", "fantasyprosId", "fantasypros");

  // Calculate match confidence and variance
  const entries = Array.from(master.values());
  for (const entry of entries) {
    entry.matchConfidence = entry.sources.length / 3;
    entry.status = entry.sources.length === 3 ? "matched_all"
      : entry.sources.length === 2 ? "matched_two"
      : "matched_one";

    const ranks = [entry.espnRank, entry.fleaflickerRank, entry.fantasyprosRank]
      .filter((r): r is number => r !== null);
    if (ranks.length > 0) {
      entry.averageRank = Math.round(ranks.reduce((a, b) => a + b, 0) / ranks.length);
      if (ranks.length > 1) {
        entry.rankVariance = Math.round(
          Math.sqrt(ranks.reduce((sum, r) => sum + Math.pow(r - entry.averageRank, 2), 0) / ranks.length) * 100
        ) / 100;
      }
    } else {
      entry.averageRank = 300;
    }
  }

  // Sort by ESPN rank (primary), fall back to average rank
  entries.sort((a, b) => {
    const aRank = a.espnRank ?? a.averageRank ?? 999;
    const bRank = b.espnRank ?? b.averageRank ?? 999;
    return aRank - bRank;
  });

  console.log(`Merge: ${entries.length} total players`);
  console.log(`  matched_all: ${entries.filter((e) => e.status === "matched_all").length}`);
  console.log(`  matched_two: ${entries.filter((e) => e.status === "matched_two").length}`);
  console.log(`  matched_one: ${entries.filter((e) => e.status === "matched_one").length}`);

  return entries;
}

// Run
const merged = merge();
fs.writeFileSync("merged-all.json", JSON.stringify(merged, null, 2));
console.log(`Wrote ${merged.length} entries to merged-all.json`);
```

## Potential Issues & Mitigations

1. **FantasyPros HTML structure changes**: The Cheerio scraper has a fallback JSON parsing path. If both fail, inspect the page HTML and adjust selectors.
2. **FleaFlicker API requires auth**: If the API returns 400/403, the merge script gracefully falls back to whatever cached data exists. User can manually provide FF data if needed.
3. **ESPN input missing**: Without `data/cache/espn-input.*`, the pipeline runs with only FF + FP data. ESPN being the primary sort key means the merge uses average rank when ESPN is absent.
4. **2026 bye weeks unknown**: Default to 7 for all teams until the schedule is released, then update `byeWeeks` in `transform-data.js`.
